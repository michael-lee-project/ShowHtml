公益宝系统
村社公益基金平台 · 5 万+ 村社基金 · 湖北等省使用 · 体制内最大
公益宝是国内最大的村社公益基金管理系统，接入 5 万+ 村社基金账户，但大量"僵尸账户"因无利益分配而失活。慈商联营通过 0 管理费分润 + 村社基金自动归集能力，可为公益宝账户重新注入活力。
🔌 API: REST + Webhook 📊 数据: 基金账户 / 流水 / 捐赠人 🔒 认证: OAuth 2.0 ⏰ 同步: 实时 🛠️ 对接方: 中科大一附院公益宝团队
🛠️ 接口开发中
预计 2026 Q4 上线
📋
慈善中国 · 全国慈善信息公开平台
民政部 · 法定信息披露平台 · 必对接
民政部主管的法定慈善信息公开平台，所有慈善组织年度报告必须在此公示。法定对接义务，慈商联营自动生成符合民政部规范的 XML 报告文件并按季度推送。
🔌 API: XML / SOAP 📊 数据: 慈善组织年报 / 募捐信息 / 审计 🔒 认证: 数字证书 ⏰ 同步: 季度
✅ 已就绪
2026 Q1 已完成对接
💳
中国银行公益专户系统
慈善资金专户管理 · 财务合规硬要求
合作银行（不绑定特定银行）的公益资金专户管理系统，按《慈善法》要求必须专户管理、专款专用、独立核算。慈商联营分润流水 + 票据 + 审计报告自动同步到银行。
🔌 API: REST + ISO 20022 📊 数据: 资金流水 / 余额 / 票据 🔒 认证: 银行级 UKey ⏰ 同步: 实时（T+0）
✅ 已就绪
2026 Q2 已完成 · 工商银行 / 招商银行 / 平安银行
🏢
国家政务服务平台 · 信用核查
企业资质核验 · 法定代表人 · 信用记录
企业入驻慈商联营时，需要核验营业执照、法人身份、信用记录等。慈商联营接入国家政务服务平台的"信用核查 API"，企业提交资料后 5 秒内自动完成全量核验，无需人工审核。
🔌 API: REST + OAuth 2.0 📊 数据: 工商 / 法人 / 信用 / 失信 🔒 认证: 政务证书 ⏰ 同步: 实时
🛠️ 接口开发中
预计 2026 Q3 上线

全国志愿服务信息系统
志愿者时长国家认证 · 民政部主管
志愿者服务时长需要在该平台同步认证才能计入个人档案。慈商联营每次志愿者完成任务后，自动同步时长 + 服务记录到国家系统，志愿者无需重复登记。
🔌 API: REST 📊 数据: 志愿者档案 / 服务时长 / 培训记录 🔒 认证: 志愿者注册号 ⏰ 同步: 实时
