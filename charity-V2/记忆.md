## 2026-06-18

### 驿站（H5 移动端）功能补全

**背景：** web 端已有完整的驿站功能（`mall/stations.html` 列表页 + `mall/station-detail.html` 详情页），H5 移动端此前没有。

**交付物：**

| 文件 | 说明 |
|------|------|
| `m/stations.html` | 驿站列表页：Hero Banner、4格统计条、搜索框、横滑筛选标签、单列驿站卡片、申请开设驿站 CTA、Tabbar |
| `m/station-detail.html` | 驿站详情页：Hero + 年度荣誉标签、三格操作条（导航/电话/收藏）、基本信息面板、月度贡献板块（销售额/基金归集/人次/自提单）、服务项目网格、驿站热卖商品列表、社区基金归集提示、活动卡片（含报名按钮四种状态）、地图占位、底部悬浮预约条 |
| `m/mall.html` | 商城首页新增驿站入口：主卡片（暖橙色）+ 附近驿站横滑推荐卡 |

**设计规范：**
- 主题色：`--h5-org: #C4842A`（公益橙），与 web 端驿站暖橙系保持一致
- 适配 390×844（iPhone 12），max-width 390 桌面模拟
- GSAP 入场动画（hero / 统计 / 卡片 / 活动 stagger）
- Tabbar 复用商城现有结构，固定在底部

**参考来源：** `mall/stations.html` + `mall/station-detail.html`（web 端）

**用户决策：**
- 驿站定义 = 社区领取点（用户到附近驿站自提捐赠物资），非仓储/物流/商家入驻
- 每次做新原型页面时，先问用户要不要走 taste-skill

## 2026-06-12

### 会议纪要核心结论

详见 `纪要.md`。关键决策：
- 平台定位：物资捐赠（非资金公募），商业反哺公益
- 5 种角色：爱心企业 / 慈善总会+基金会 / 志愿者 / 受捐者
- 慈善驿站：线下空间 + 严选商品 + 物资自提 + 社区服务
- 严选模式：厂家直供，避免比价和假货
- 村社基金：消费利润 5% 自动归集，0 管理费
- 合规要求：捐赠票据、审计公示、受捐人隐私隔离
